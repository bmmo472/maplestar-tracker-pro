# MapleStar Tracker Pro - 使用說明

楓星經驗追蹤工具。每秒抓取畫面上的 EXP 區域跑 OCR，計算經驗速率與升等時間。

---

## 系統需求

- Windows 10 / 11（64-bit）
- 4 GB RAM 以上
- 解壓後約 1 GB 硬碟空間
- **不需要顯卡**，CPU 推論即可

---

## 第一次啟動

### 解壓

下載 `MapleStarTrackerPro_v1.2.0.zip` 後，**整個資料夾解壓到任何位置**（建議桌面或 D 槽）。

不要單獨把 `.exe` 拉出來執行，會找不到相依檔案。

### 防毒可能阻擋

PyInstaller 打包的 EXE 經常被 Windows Defender 或第三方防毒誤判。

如果開不起來：
1. 開檔案總管找到 `MapleStarTrackerPro.exe`
2. 右鍵 → 內容 → 一般 → 勾選「解除封鎖」→ 套用
3. 或加入防毒白名單

不放心可以從原始碼跑（看 `INSTALL_FROM_SOURCE.md`）。

### 啟動

雙擊 `MapleStarTrackerPro.exe`。首次啟動會載入 OCR 模型，**等 5-10 秒**才會出現主視窗。

## 設定流程（首次使用必做）

### 步驟 1：開啟楓星

先把楓星遊戲開起來。**不需要登入角色**，但畫面要能看到經驗值跟等級。

建議停在城鎮或地圖選擇畫面，避免戰鬥動畫干擾。

### 步驟 2：選擇目標視窗

切到 **設定** 分頁。

「目標視窗」下拉選單會顯示所有開啟的視窗。楓星視窗會自動排在最上面、帶 `★` 標記：

```
★ MapleStory Worlds-楓星  (1920×1080)
File Explorer
Chrome
...
```

通常**已經自動選好**，不用動。如果有多開可以切換。

如果沒看到 ★ 楓星：
- 確認楓星真的開著（不要最小化到工具列）
- 點「重新整理」按鈕重新掃描

### 步驟 3：框選 EXP 區域

點「框選 EXP 區域」按鈕。

跳出來的視窗會擷取你的楓星畫面，**用滑鼠拖拉框出「EXP 數字 + 百分比」那一塊**。

範圍要訣：
- 框緊一點，**只包含 EXP 文字本身**
- 不要框到旁邊的圖示或無關文字
- 框的長寬比例不對沒關係，OCR 會自動處理

框完按確定。下方會顯示「已設定：W×H @ (X, Y)」綠字。

### 步驟 4：框選等級區域（推薦做，可省略）

點「框選等級區域」按鈕。

框出「Lv 158」那一塊（左上角等級顯示位置）。

- 不框等級的話：要回到 **即時追蹤** 分頁手動點「校正目前等級」輸入數字
- 框了等級：程式自動辨識，升級時也會自動更新

### 步驟 5：（可選）調整更新頻率

預設每 1 秒抓一次畫面跑 OCR。可以調 2 / 3 / 5 / 10 秒節省 CPU。

一般 1 秒最好，反應最即時。

## 開始追蹤

回到 **即時追蹤** 分頁，點「開始追蹤」。

正常情況下：

```
LEVEL                                  PROGRESS
Lv 158                                 72.01%
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

           CURRENT EXP
        138,126,520

累計時間  本次累積   升下一級    剩餘 EXP    辨識狀態
00:01:06  137,709K   24 秒      53,683K    已採用

1 分速率   5 分累積    5 分推估   10 分累積   10 分推估
600,000   2,800,000  ~3.00M    5,400,000  ~6.00M
```

### 各區塊意思

| 區塊 | 意思 |
|---|---|
| **LEVEL** | 目前等級 |
| **PROGRESS** | 等級內進度百分比 |
| **CURRENT EXP** | 目前 EXP 數值 |
| **累計時間** | 從點開始追蹤到現在多久 |
| **本次累積** | 這次 session 打了多少 EXP |
| **升下一級** | 依目前速率還要多久升級 |
| **剩餘 EXP** | 升下一級還差多少 |
| **辨識狀態** | OCR 內部狀態（取樣中 / 已採用 / 共識失敗） |
| **1 分速率** | 上一個完整 60 秒打的 EXP |
| **5 分累積** | 上一個完整 5 分鐘打的 EXP |
| **5 分推估** | 用目前速率推估 5 分鐘能打多少（~代表預估） |
| **10 分累積** | 上一個完整 10 分鐘打的 EXP |
| **10 分推估** | 用目前速率推估 10 分鐘能打多少 |

### 為什麼一開始「辨識狀態」是「取樣中 1/5」？

工具為了**準確**設計成「5 秒取一次共識值才更新」。

- 每秒抓畫面跑 OCR，但**累積到 5 筆**才檢查共識
- 5 筆 OCR 中央 3 筆數字夠接近 → 採用
- 不夠接近 → 整批丟棄等下輪
- 防止 OCR 偶爾讀錯（例如 138,126,520 看成 138,131,000）讓累積值亂跳

慢一點但準。實際打怪過 5-10 秒就會穩定每 5 秒更新一次。

## 懸浮視窗

點主視窗右上「懸浮視窗」按鈕，會跳出小視窗。

特性：
- 永遠在最上層（覆蓋遊戲畫面）
- 可拖動位置
- **右下角拖拉可以調整大小**
- 右鍵選單調整透明度（70% / 80% / 92% / 100%）
- 右鍵選單有「尺寸預設」一鍵切換大小

打怪時開啟，瞄一眼就看到速率跟 ETA。

---

## 重置 / 停止

- **停止**：暫停追蹤，數據保留。可以再點「開始追蹤」繼續。
- **重置**：清空所有累積數據（時間、累積 EXP、速率）。重新開始 session。

## 常見問題

### Q：累積 EXP 一直顯示 0

確認：
1. 你**有打怪**嗎？沒打怪不會增加
2. 「辨識狀態」顯示什麼？
   - `取樣中 N/5` → 正常，等 5 秒會更新
   - `共識失敗` → OCR 抖動太大，看看 EXP 區域是不是框錯
   - `已採用` → 正常運作

### Q：等級顯示錯誤

兩種情況：
- **手動設定的話**：到「即時追蹤」分頁「校正目前等級」改數字
- **OCR 辨識的話**：到「設定」分頁重新「框選等級區域」，框緊一點

### Q：升下一級時間不準

升下一級 = 剩餘 EXP / 目前速率。

- 剛開始追蹤前 5 分鐘速率還不穩，ETA 會跳
- 跑 10 分鐘後就會收斂

### Q：CPU 占用很高

OCR 每秒跑一次比較吃 CPU。可以：
- 「設定」分頁把更新頻率改 2-3 秒
- 關閉懸浮視窗（如果不需要）

### Q：可以多開楓星追蹤多個帳號嗎

可以。再開一個 `MapleStarTrackerPro.exe`，每個 instance 選不同的視窗即可。

### Q：升級時會自動偵測嗎

會。等級 ROI 框好的話會自動偵測。

升級瞬間「本次累積」會補上「上一級剩餘 EXP + 新等級已得 EXP」，速率計算不受影響。

### Q：可以匯出資料嗎

目前版本沒有。如果需要可以告訴作者。

## 設定檔位置

設定儲存在：

```
%APPDATA%\MapleStarTrackerPro\settings.json
```

包含：
- 上次選擇的視窗
- 框選的 EXP / 等級 ROI
- 手動設定的等級
- 浮窗位置 / 大小 / 透明度

要重置所有設定就刪除這個檔案。
